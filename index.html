Abstract

We present PHOTONS (Pose-Free Human-Centric Poto-Realistic Real-Time Novel View Synthesis from Sparse Views), a real-time framework for novel view synthesis without requiring camera calibration. PHOTONS faithfully renders dynamic human bodies amid complex backgrounds, including interactive object manipulation and fine-grained details, while maintaining 25 FPS throughput on commodity GPU like NVIDIA RTX 4090.
